Installing
----------

.. code:: bash

   pip install shapeways

Documentation
-------------

Documentation can be found at `ReadTheDocs <http://python-shapeways.readthedocs.org>`_.

Building Documentation
~~~~~~~~~~~~~~~~~~~~~~

Once built the docs can be accessed from ``/docs/_build/html/index.html``

.. code:: bash

    git clone git://github.com/Shapeways/python-shapeways.git
    cd python-shapeways
    make docs
